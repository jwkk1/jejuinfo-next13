> next13 신규문법 app 디렉토리활용, visitJeju API를 이용한 제주 관광정보 웹


## 배포

http://jeju-6-env.eba-qmgujw3b.ap-northeast-2.elasticbeanstalk.com/


## 구현기능

* 회원가입 / 로그인 / 로그아웃
* visitJeju API 연동
* 즐겨찾기 저장, mongoDB 연동
* SNS공유 및 지도 연동
* 여행일정 플래너 (개발중)
